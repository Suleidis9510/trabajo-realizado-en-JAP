Introducción al Desarrollo Web:

1. ¿Qué es el Desarrollo Web?

El desarrollo web es el proceso de creación y mantenimiento de sitios web. Incluye aspectos como el diseño web, desarrollo de contenido, scripting del lado del cliente/servidor y configuración de seguridad de la red, entre otros.

2. Componentes del Desarrollo Web

Front-End (Cliente)

  • HTML (HyperText Markup Language): Lenguaje de marcado utilizado para estructurar y presentar el contenido en la web.
  • CSS (Cascading Style Sheets): Lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML.
  • JavaScript: Lenguaje de programación que permite crear contenido dinámico e interactivo en la web.

Back-End (Servidor)

  • Lenguajes de programación: PHP, Python, Ruby, Java, Node.js.
  • Bases de datos: MySQL, PostgreSQL, MongoDB.
  • Servidores: Apache, Nginx.

3. Desarrollo Front-End

HTML

Estructura básica de un documento HTML. Etiquetas comunes:

  • <html>, <head>, <body>
  • <div>, <span>
  • <a>, <img>
  • <h1> a <h6>, <p>, <ul>, <ol>, <li>, <form>, <input>, <button>

CSS

Selectores y propiedades básicas: color, font-size, margin, padding, border. Modelo de caja (box model): cómo los elementos HTML se representan en el navegador. Diseño responsivo: uso de media queries para adaptar el diseño a diferentes tamaños de pantalla.

JavaScript

Variables, funciones y eventos. Manipulación del DOM (Document Object Model). Librerías y frameworks populares: jQuery, React, Angular, Vue.js.

4. Desarrollo Back-End

Lenguajes de programación

PHP, Python, Ruby, Java, Node.js.

Bases de datos

SQL (Structured Query Language): Lenguaje estándar para gestionar bases de datos relacionales. NoSQL: Bases de datos no relacionales como MongoDB, diseñadas para datos flexibles y no estructurados.

Servidores Web

Apache: Uno de los servidores web más utilizados. Nginx: Conocido por su alto rendimiento y bajo consumo de recursos.

5. Herramientas y Tecnologías

Sistemas de Control de Versiones

Git, GitHub.

Entornos de Desarrollo Integrados (IDE)

Visual Studio Code, Sublime Text, Atom.

Frameworks y Librerías

  • Front-End: Bootstrap, Foundation.
  • Back-End: Django, Flask, Ruby on Rails, Express.js.

6. Metodologías de Desarrollo

Desarrollo Ágil: Scrum, Kanban. Desarrollo en Cascada (Waterfall).

7. Buenas Prácticas

  • Escribir código limpio y comentado.
  • Seguir estándares web.
  • Optimización para motores de búsqueda (SEO).
  • Seguridad web: Protección contra ataques comunes como SQL injection, Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF).

8. Recursos de Aprendizaje

  • freeCodeCamp
  • Codecademy
  • Udemy
  • Coursera

Conclusión

El desarrollo web es un campo amplio y en constante evolución que combina creatividad y habilidades técnicas. Aprender las bases del desarrollo web es el primer paso para construir aplicaciones web robustas y atractivas.

Semana 2: Introducción a HTML

1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. HTML describe la estructura de una página web mediante el uso de elementos etiquetados.

2. Estructura Básica de un Documento HTML

<!DOCTYPE html>
        <html lang="es">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título de la página</title>
        </head>
        <body>
        <h1>Encabezado Principal</h1>
        <p>Párrafo de texto.</p>
        </body>
        </html>

3. Elementos Comunes de HTML

  • <html>: Elemento raíz que envuelve todo el contenido de la página.
  • <head>: Contiene metadatos, el título de la página y enlaces a recursos externos como hojas de estilo.
  • <title>: Define el título de la página que aparece en la barra del navegador.
  • <body>: Contiene el contenido visible de la página web.
  • <h1> a <h6>: Encabezados de diferentes niveles, siendo <h1> el más importante y <h6> el menos importante.
  • <p>: Define un párrafo de texto.
  • <a>: Define un enlace.
  • <img>: Inserta una imagen en la página.
  • <ul>: Define una lista desordenada.
  • <ol>: Define una lista ordenada.
  • <li>: Define un ítem de lista.
  • <div>: Define una sección o contenedor genérico en la página.
  • <span>: Define una sección en línea o contenedor genérico.

4. Atributos de HTML

Los elementos HTML pueden tener atributos que proporcionan información adicional sobre el elemento. Los atributos se escriben dentro de la etiqueta de apertura y generalmente vienen en pares nombre/valor como nombre="valor".

  • class: Asigna una o más clases a un elemento, que pueden ser utilizadas para aplicar estilos CSS.
  • id: Asigna un identificador único a un elemento, que puede ser utilizado para aplicar estilos CSS o manipular el elemento con JavaScript.
  • href: Especifica la URL de la página a la que enlaza un enlace.
  • src: Especifica la URL de la imagen a mostrar.
  • alt: Proporciona un texto alternativo para una imagen.

5. Ejemplo Completo

A continuación se muestra un ejemplo completo de una página HTML con algunos de los elementos y atributos mencionados: 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>Bienvenidos a Mi Página</h1>
    <p>Este es un ejemplo de una página HTML.</p>
    <img src="ruta/a/mi-imagen.jpg" alt="Descripción de la imagen">
    <p>Visita <a href="https://www.ejemplo.com">este enlace</a> para más información.</p>
</body>
</html>

Semana 3: Introducción a CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. CSS controla el aspecto visual de la página web, incluyendo colores, fuentes, márgenes, bordes y disposición del contenido.

2. Cómo Incluir CSS en HTML

  • Inline CSS: Utilizando el atributo style dentro de un elemento HTML.
  • Internal CSS: Usando una etiqueta <style> dentro del <head> del documento HTML.
  • External CSS: Enlazando un archivo CSS externo utilizando la etiqueta <link> dentro del <head>.

3. Selectores y Propiedades Básicas de CSS

CSS utiliza selectores para aplicar estilos a elementos HTML. A continuación, se muestran algunos selectores comunes y propiedades básicas:

  • Selector de Elemento: Aplica estilos a todos los elementos de un tipo específico. 
    p {
    color: blue;
}
  • Selector de Clase: Aplica estilos a todos los elementos con una clase específica. 
    .mi-clase {
    font-size: 20px;
}
  • Selector de ID: Aplica estilos a un elemento con un ID específico. 
    #mi-id {
    background-color: yellow;
}

4. Modelo de Caja (Box Model)

El modelo de caja en CSS describe cómo se representan los elementos en el navegador. Cada elemento se representa como una caja rectangular, que incluye:

  • Contenido: El contenido del elemento, como texto o imágenes.
  • Relleno (Padding): Espacio entre el contenido y el borde.
  • Borde (Border): Línea que rodea el relleno.
  • Margen (Margin): Espacio entre el borde y otros elementos. 
div {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    margin: 15px;
}

5. Ejemplo Completo

A continuación se muestra un ejemplo completo de cómo utilizar CSS para estilizar un documento HTML: 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Estilizada</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f4f4f4;
            color: #333;
        }
        h1 {
            color: #3498db;
        }
        .contenido {
            background-color: #fff;
            padding: 20px;
            border-radius: 8px;
            box-shadow: 0 0 10px rgba(0,0,0,0.1);
        }
        .contenido p {
            line-height: 1.6;
        }
    </style>
</head>
<body>
    <h1>Bienvenidos a Mi Página Estilizada</h1>
    <div class="contenido">
        <p>Este es un ejemplo de una página HTML con CSS aplicado para mejorar su apariencia visual.</p>
    </div>
</body>
</html>

Semana 4: Introducción a JavaScript (DOM, HTTP y Fetch)

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear contenido dinámico e interactivo en las páginas web. Se ejecuta en el navegador del cliente, lo que lo hace ideal para realizar tareas que requieran una respuesta rápida y sin necesidad de recargar la página.

2. Manipulación del DOM

El DOM (Document Object Model) es una interfaz de programación que permite a los scripts actualizar el contenido, la estructura y el estilo de un documento mientras se está viendo. Con JavaScript, puedes seleccionar y manipular elementos del DOM para cambiar la apariencia y el comportamiento de una página web.

Selección de Elementos

// Seleccionar un elemento por su ID

const elemento = document.getElementById('mi-id');


// Seleccionar elementos por su clase

const elementos = document.getElementsByClassName('mi-clase');


// Seleccionar elementos utilizando un selector CSS

const parrafos = document.querySelectorAll('p');

Manipulación de Elementos



// Cambiar el contenido de texto de un elemento

elemento.textContent = 'Nuevo contenido';


// Cambiar el estilo de un elemento

elemento.style.color = 'blue';


// Añadir una clase a un elemento

elemento.classList.add('nueva-clase');


// Crear un nuevo elemento y añadirlo al DOM

const nuevoElemento = document.createElement('div');
nuevoElemento.textContent = 'Soy un nuevo div';
document.body.appendChild(nuevoElemento);

3. Solicitudes HTTP

JavaScript permite realizar solicitudes HTTP para comunicarse con servidores y recuperar datos sin necesidad de recargar la página. Esto se puede hacer utilizando objetos como XMLHttpRequest o la API Fetch.

XMLHttpRequest

const xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.ejemplo.com/datos', true);
xhr.onload = function() {
    if (xhr.status === 200) {
        console.log(xhr.responseText);
    }
};
xhr.send();

4. API Fetch

La API Fetch es una forma moderna y más sencilla de realizar solicitudes HTTP en JavaScript. Utiliza promesas para manejar la asincronía. 

fetch('https://api.ejemplo.com/datos')
    .then(response => response.json())
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error('Error:', error);
    });

5. Ejemplo Completo

A continuación se muestra un ejemplo completo que combina la manipulación del DOM y el uso de la API Fetch para recuperar y mostrar datos desde una API. 

<!DOCTYPE html>
    <html lang="es">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo Completo de JavaScript</title>
    <style>
        body {
            font-family: Arial, sans-serif;
        }
        .usuario {
            background-color: #f4f4f4;
            padding: 10px;
            margin: 10px 0;
            border-radius: 4px;
        }
    </style>
</head>
<body>
    <h1>Lista de Usuarios</h1>
    <div id="usuarios"></div>

    <script>
        
// Función para obtener datos de una API y mostrarlos en el DOM

        function cargarUsuarios() {
            fetch('https://jsonplaceholder.typicode.com/users')
                .then(response => response.json())
                .then(data => {
                    const contenedorUsuarios = document.getElementById('usuarios');
                    data.forEach(usuario => {
                        const divUsuario = document.createElement('div');
                        divUsuario.classList.add('usuario');
                        divUsuario.textContent = `Nombre: ${usuario.name}, Email: ${usuario.email}`;
                        contenedorUsuarios.appendChild(divUsuario);
                    });
                })
                .catch(error => {
                    console.error('Error:', error);
                });
        }

      
// Cargar usuarios al cargar la página

        window.onload = cargarUsuarios;
  </script>
</body>
</html>
